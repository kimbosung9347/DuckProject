## ItemBox System

아이템 상자(루팅) 전용 시스템으로, 확률·시간·상태 기반 설계를 중심으로 구성된 단일 컴포넌트입니다.

### 특징
- 아이템 **등급 확률(Normal ~ Legend)** 및 **드랍 개수 확률(1~10개)**을 데이터 기반으로 설정
- 슬롯 단위로 `NotSearch → Searching → Complete` 상태를 가지는 **탐색 진행 시스템**
- 일반 랜덤 상자와 퀘스트/연출용 **Force Item Box**를 동일 구조로 처리

### 주요 구성
- **ItemBoxBase**  
  아이템 상자의 핵심 로직을 담당하며, 아이템 생성·확률 계산·탐색 상태·슬롯 관리까지 포함하는 루팅 시스템의 중심 컴포넌트

### 시스템 설명
- 상자는 내부적으로 고정 슬롯 배열을 가지며, 각 슬롯은 독립적인 탐색 상태와 시간을 가짐
- 탐색 중에는 슬롯별로 시간이 흐르며, 완료 시 아이템이 공개되고 사운드·UI가 갱신됨
- `Force` 타입 상자는 지정된 아이템 리스트를 우선 배치하여 퀘스트 보상, 연출용 루팅을 안정적으로 처리 가능
- 아이템 생성은 `GameInstance`의 팩토리 API를 통해 이루어져 무기·방어구·부착물·소비·재료 아이템을 동일한 흐름으로 생성
