## UI Item Slot System

아이템 슬롯 UI는 **입력 처리 / 시각 표현 / 데이터 표시**를 명확히 분리하고,  
프리팹 + 중첩 프리팹 구조로 다양한 슬롯 타입(인벤토리, 장비, 퀵슬롯, 상점 등)을 공통 시스템으로 처리하도록 설계되었습니다.

- **SlotDesc**  
  슬롯 하단 정보 영역을 담당하며 아이템 이름·개수·내구도 게이지·부착물 상태를 상황에 따라 전환 표시하는 UI 표현 컴포넌트

- **SlotInputBase**  
  Hover·Click·Drag 입력을 공통으로 처리하고 SlotController에 슬롯 타입·인덱스·아이템 정보를 전달하는 입력 베이스 클래스

- **UIItemSlotBase**  
  슬롯의 시각적 표현(아이콘, 등급 테두리, 선택 상태, 첨부 정보)을 관리하며 실제 슬롯 타입별 UI의 공통 기반 클래스

### 설계 특징
- 모든 슬롯 입력은 `SlotInputBase`에서 통합 처리하여 슬롯별 로직 중복을 제거
- 슬롯의 시각 정보와 입력 로직을 분리해 유지보수성과 확장성 확보
- 아이템 슬롯들은 **프리팹 + 중첩 프리팹 구조**로 제작되어  
  인벤토리 / 장비 / 퀵슬롯 / 상점 / 창고 등 다양한 UI에 동일한 슬롯 구조를 재사용
- SlotController를 중심으로 드래그·드롭·우클릭 액션 등 슬롯 간 상호작용을 중앙 관리
