## Duck Components

AI와 Player가 공통으로 사용하는 캐릭터 핵심 컴포넌트들로,
각 기능을 역할 단위로 분리하여 조합·확장 가능한 구조로 설계했습니다.

- **DuckAbility**  
  캐릭터의 이동·사격·방어·스탯·수용량 능력치를 통합 관리하고 보정값을 계산하는 핵심 능력치 시스템

- **DuckAiming**  
  조준 및 반동 상태를 관리하며 조준 진행도에 따른 명중 보정을 능력치 시스템과 연동

- **DuckAnimation**  
  능력치 정보를 기반으로 이동·공격·구르기 애니메이션을 절차적으로 제어하는 컴포넌트

- **DuckAppearance**  
  무기·방어구·소비 아이템을 캐릭터 소켓에 장착/해제하는 외형 관리 컴포넌트

- **DuckAttack**  
  무기와 조준 정보를 기반으로 사격, 반동, 명중 판정을 처리하는 전투 공격 컴포넌트

- **DuckMeleeAttack**  
  선딜·판정·쿨타임을 상태 기반으로 관리하는 근접 공격 컴포넌트

- **DuckBuff**  
  시간 기반 및 영구 버프를 관리하고 능력치 보정을 적용하는 버프 시스템

- **DuckDead**  
  캐릭터 사망 상태 전환과 사망 이펙트·사운드를 처리하는 컴포넌트

- **DuckDetected**  
  시야 시스템에 따라 캐릭터 렌더러와 HP UI 표시를 제어하는 감지 컴포넌트

- **DuckEquip**  
  무기·방어구·백팩 장착과 능력치·외형 동기화를 관리하는 장비 시스템

- **DuckHit**  
  피격 판정과 부위별 방어력을 고려한 데미지 계산을 담당하는 컴포넌트

- **DuckHpbar**  
  체력 UI를 제어하고 피격 시 감소 연출을 처리하는 HP 바 컴포넌트

- **DuckLocomotion**  
  이동·회전·구르기를 NavMesh와 충돌 검사 기반으로 처리하는 이동 컨트롤러

- **DuckMeshSetter**  
  캐릭터 메쉬와 색상을 관리하며 런타임 외형 변경을 지원하는 컴포넌트

- **DuckSpeechBubble**  
  타이핑 효과 기반 말풍선을 표시하는 캐릭터 UI 컴포넌트

- **DuckStat**  
  HP·MP 관리 및 피격·회복·사망 판정을 처리하는 스탯 시스템

- **DuckState**  
  캐릭터 행동 상태를 관리하고 상태별 입력 제한과 능력치 변화를 제어하는 상태 머신

- **DuckStorage**  
  인벤토리와 소비 아이템 사용 흐름을 관리하고 상태·버프와 연동하는 저장소 시스템
