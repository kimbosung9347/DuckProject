## Player Components

Duck 공통 컴포넌트를 기반으로 플레이어 전용 입력·상태·시스템을 확장했으며 
주로 UI·세이브 흐름을 확장시킨 컴포넌트 집합입니다.

- **PlayerController**  
  플레이어 입력을 수집해 이동·조준·전투·UI 전환 흐름을 총괄 제어하는 메인 컨트롤러

- **PlayerState**  
  조준·재장전·구르기·사망 등 상태 전이를 관리하고 카메라·UI·스태미나 동작을 연동

- **PlayerStat**  
  체력·스태미나·허기·갈증 스탯과 난이도 보정·버프·HUD 갱신을 통합 관리

- **PlayerSight**  
  주·야 및 버프에 따른 시야(FOV)·다크 영역을 제어하고 원형 감지로 적 노출을 처리

- **PlayerSkill**  
  적 탐지·구르기 스킬을 실행하고 쿨타임·스태미나·버프·HUD를 연동 관리

- **PlayerAiming**  
  마우스 위치와 상태를 기반으로 조준 방향·목표 지점을 계산하는 조준 로직

- **PlayerAttack**  
  사격 처리와 반동·카메라 임펄스·맵 알림을 포함한 플레이어 공격 컴포넌트

- **PlayerEquip**  
  무기·방어구·부착물 장착과 재장전·탄약·UI 동기화를 담당하는 장비 시스템

- **PlayerStorage**  
  인벤토리 용량·무게·아이템 추적·탄약/소모품 병합·장전 연동을 담당하는 저장소

- **PlayerQuickSlot**  
  소비 아이템 퀵슬롯 매핑과 사용 조건 판별·슬롯 교체·UI 갱신을 처리

- **PlayerQuest**  
  퀘스트 수락·진행·완료 상태 관리와 킬·전달·레벨 조건 및 보상 처리

- **PlayerGrow**  
  경험치·레벨·성장 수치를 관리하는 플레이어 성장 전용 컴포넌트

- **PlayerTime**  
  플레이 시간과 게임 시간을 분리 관리하고 주·야 전환에 따른 맵·시야·UI를 연동

- **PlayerDetector**  
  상호작용 대상 탐지와 인터랙션 가능 상태를 판별하는 근접 탐지 컴포넌트

- **PlayerEnemyDetector**  
  범위 기반 적 탐지 스킬 전용 처리 및 HUD 표시를 담당

- **PlayerMeshSetter**  
  외형 데이터를 기반으로 메시를 생성·교체하고 저장/로드와 연동

- **PlayerDead**  
  사망 처리와 난이도별 아이템 드랍·맵 데이터 반영·복귀 흐름을 관리

- **PlayerDifficult**  
  현재 난이도 상태와 피해 계수 변환을 관리하는 난이도 컴포넌트

- **PlayerSave**  
  출격·철수·사망 시 플레이어·맵 데이터를 비동기 저장해주는 컴포넌트
  
- **PlayerUIController** 
  플레이어와 연관이 있는 전체 UI를 중앙에서 중계·전환·갱신하는 UI 허브
