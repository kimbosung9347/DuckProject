## AI Components

AI 전용 캐릭터 로직을 담당하는 컴포넌트들로, 
공통 Duck 구조 위에 AI기능에 필요한 기능들을 확장시켰습니다.

- **AiController**  
  순찰·추적·전투 상태에 따른 행동 흐름을 제어하는 AI 컨트롤러

- **AiAbility**  
  AI 캐릭터의 기본 능력치와 난이도 보정 값을 적용시켜주는 컴포넌트

- **AiAiming**  
  타겟 Transform을 기준으로 조준 방향과 목표 지점을 자동 계산하여
  AI가 지속적으로 목표를 추적·조준하도록 하는 컴포넌트

- **AiDead**  
  AI 사망 시 장착 장비·탄약·드랍 아이템을 생성하고
  퀘스트·업적 처리까지 포함한 사망 후 처리를 담당하는 컴포넌트

- **AiDetected**  
  플레이어에게 감지되었을 때, AI 캐릭터의 렌더러와 HP UI를 노출하는 감지 상태 관리 컴포넌트

- **AiDetector**  
  시야각·거리·라인 오브 사이트 기반 탐지와 공격 가능 대상 선별을 수행하는 AI 감지 시스템

- **AiHit**  
  피격 시 공격자를 감지 시스템에 전달하여 AI가 피격 원인을 인식 및 대응하도록 확장한 컴포넌트

- **AiStorage**  
  AI가 생성이 될 때, 가중치 기반 랜덤 장비·탄약을 생성해주는 컴포넌트
