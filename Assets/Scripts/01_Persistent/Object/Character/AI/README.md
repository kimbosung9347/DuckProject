## AI Components

AI 전용 캐릭터 로직을 담당하는 컴포넌트들로,
공통 Duck 구조 위에 탐지·판단·전투·드랍 흐름을 확장하여
자율적인 전투 행동을 수행하도록 설계했습니다.

- **AiAbility**  
  AI 캐릭터의 기본 능력치와 난이도 보정 값을 적용하고,
  감지·전투 시스템에서 사용하는 능력치 정보를 제공하는 컴포넌트

- **AiAiming**  
  타겟 Transform을 기준으로 조준 방향과 목표 지점을 자동 계산하여
  AI가 지속적으로 목표를 추적·조준하도록 하는 컴포넌트

- **AiAttack**  
  탄약을 소지하지 않는 AI 특성에 맞춰
  자동 재장전 기반 사격 로직을 처리하는 AI 전용 공격 컴포넌트

- **AiController**  
  NavMesh 기반 이동과 조준 방향을 연동하여
  순찰·추적·전투 상태에 따른 행동 흐름을 제어하는 AI 컨트롤러

- **AiDead**  
  AI 사망 시 장착 장비·탄약·드랍 아이템을 생성하고
  퀘스트·업적 처리까지 포함한 사망 후 처리를 담당하는 컴포넌트

- **AiDetected**  
  플레이어에게 감지되었을 때만
  AI 캐릭터의 렌더러와 HP UI를 노출하는 감지 상태 관리 컴포넌트

- **AiDetector**  
  시야각·거리·라인 오브 사이트 기반 탐지와
  공격 가능 대상 선별을 수행하는 AI 감지 시스템

- **AiHit**  
  피격 시 공격자를 감지 시스템에 전달하여
  AI가 피격 원인을 인식하고 대응하도록 하는 컴포넌트

- **AiStorage**  
  가중치 기반 랜덤 장비·탄약을 생성하고
  AI 사망 시 드랍 데이터로 전환하는 전용 장비/아이템 관리 컴포넌트
