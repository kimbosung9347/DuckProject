## AdornDuck

캐릭터 커스터마이징 화면에서 사용되는 오브젝트입니다.

플레이어와 동일한 **Mesh 구조를 프리팹 형태로 구성**하고 있으며,  
해당 오브젝트가 활성화될 때 현재 플레이어의 외형 정보와 동기화됩니다.

각 부위(Body, Eyes, Hair, Beak, Hand, Foot 등)는 독립적으로 교체 가능하며,  
색상 또한 `MaterialPropertyBlock`을 사용해 런타임에서 개별 적용됩니다.

커스터마이징 완료 후에는 선택된 메쉬 및 색상 정보가  
플레이어에게 적용되며, 동시에 **세이브 데이터로 동기 저장**됩니다.
