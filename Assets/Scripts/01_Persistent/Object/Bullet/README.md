## BulletProjectile

투사체(총알) 동작을 담당하는 컴포넌트입니다.  
총알은 **오브젝트 풀링 방식**으로 관리되며, 발사·충돌·반환까지의 수명 주기를 포함합니다.

- Rigidbody 기반으로 이동하며, 충돌 판정은 Trigger 방식으로 처리
- 일정 시간(`lifeTime`) 경과 시 자동으로 풀로 반환
- 발사 주체를 제외하기 위한 예외 오브젝트 리스트를 관리

피격 처리 시에는 구체 타입에 의존하지 않고  
`IDamageable`(또는 IHitTarget) 인터페이스를 통해 데미지를 전달합니다.
고로 플레이어·AI·오브젝트 등 다양한 타겟에 공통으로 사용될 수 있도록 설계되었습니다.
