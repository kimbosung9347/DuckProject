# LoadingCanvas

게임 씬 전환 및 로딩 화면 관리 시스템.

- **로딩 타입별 처리**  
  - `LobbyToHome` : 홈 씬으로 전환, 플레이어 생성, 창고 초기화  
  - `SallyForth` : 출격 로딩, 플레이어 경험치/아이템/죽은 적 처리, 비동기 저장 수행  
  - `Withdraw` : 철수 로딩, 출격 데이터 기반 저장, Achievment 처리  
  - `Dead` : 사망 로딩, 사망 처리 및 비동기 저장  
  - `End` : 메인 메뉴 이동, 플레이어 제거  

- **로딩 화면 연동**  
  - 로딩 화면별 GameObject 활성/비활성 처리  

- **비동기 작업 처리**  
  - `RunAsync`를 통해 Task 기반 저장을 Coroutine으로 처리  
  - 로딩 씬 종료 전 일정 딜레이를 적용하여 UX 보정

- **플레이어 초기화**  
  - `MakePlayer()`로 캐릭터 생성, 슬롯 컨트롤러 초기화, 비활성 상태 유지
